<h1>PIXIJS Color Picker</h1>

<p>A PIXIJS-based project offering a solution for picking colors from raster graphics. This tool is designed to work seamlessly with WebGL and TypeScript, providing an interactive and efficient color-picking experience.</p>

<h2>Installation</h2>

<p>To set up the project, you'll need Node.js installed on your machine. Follow these steps to install and run the project:</p>

<ol>
    <li>Clone the repository:
        <pre><code>git clone https://github.com/StephenUSA/pixi-colorpicker.git
cd pixi-colorpicker</code></pre>
    </li>
    <li>Install the necessary dependencies:
        <pre><code>npm install</code></pre>
    </li>
    <li>Start the development server:
        <pre><code>npm run start</code></pre>
    </li>
    <li>Open your web browser and navigate to <a href="http://localhost:8080" target="_blank">http://localhost:8080</a> to see the project in action.</li>
</ol>

<h2>Usage</h2>

<p>Once the development server is running, you can interact with the color picker by selecting any point on the graphic. The application will display the color information based on the selected pixel.</p>

<h2>Features</h2>

<ul>
    <li><strong>WebGL Rendering:</strong> Utilizes WebGL for efficient graphics rendering.</li>
    <li><strong>TypeScript Integration:</strong> The project is developed using TypeScript, ensuring type safety and robust development.</li>
    <li><strong>Interactive UI:</strong> Provides a user-friendly interface for selecting colors from graphics.</li>
</ul>
